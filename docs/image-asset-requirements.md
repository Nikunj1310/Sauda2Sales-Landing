# Image Asset Requirements

Use this checklist to add production images. Place files in `public/images/`.

## Naming Convention
- Format: `section-purpose-variant.ext`
- Examples: `hero-buyer-app-primary.jpg`, `visual-search-upload-flow.webp`
- Preferred formats: `webp` (default), `jpg` for photos, `png` only when transparency is required

## Delivery Rules
- Keep each file under `350 KB` where possible.
- Ensure no text is baked into images unless explicitly requested.
- Maintain consistent color mood: cyan/indigo/emerald accents, no heavy purple bias.
- Provide both desktop-safe crop and mobile-safe center crop area.
